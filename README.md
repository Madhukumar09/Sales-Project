# 🛒 E-Commerce Sales Analysis using SQL & Power BI

## 📌 Overview

This project is an end-to-end **E-Commerce Sales Analysis Dashboard** developed using **SQL** and **Power BI**. It analyzes sales data to uncover business insights, monitor key performance indicators (KPIs), and support data-driven decision-making.

SQL was used for data extraction and analysis, while Power BI was used to build an interactive dashboard with dynamic filters and visualizations.

## 🎯 Project Objectives

- Analyze overall sales performance.
- Monitor profit and profit margin.
- Track monthly sales trends.
- Identify top-selling products.
- Compare sales across different regions.
- Analyze category-wise performance.
- Build an interactive dashboard for business reporting.

## 📊 Business Insights

- Technology category generated the highest sales.
- West region recorded the highest revenue.
- Consumer segment contributed significantly to total sales.
- Sales increased during the final months of the year.
- Top-selling products contributed a major share of revenue.
- Profit margin across all sales is **12.47%**.
